## Monétisation (phase 2, après 100 utilisateurs)

### Modèle Freemium
Le produit de base reste **gratuit pour toujours** (open source).

**FactureFlow Pro** (SaaS hébergé) — 4,90€/mois :
- Hébergement cloud (pas de setup technique)
- Sauvegarde automatique
- Multi-device sync
- Envoi de factures par email intégré
- Relances automatiques
- Export comptable (FEC)
- Support prioritaire

**FactureFlow Business** — 9,90€/mois :
- Tout Pro +
- Multi-utilisateur (associés, comptable)
- API access
- Intégration bancaire
- Tableaux de bord avancés
- Logo et branding personnalisé sur les documents

### Projections
- 100 utilisateurs gratuits → 10-15% conversion → 10-15 abonnés Pro
- 10 × 4,90€ = **49€/mois** (premiers revenus)
- À 1000 utilisateurs : 100-150 abonnés → **490-735€/mois**
- À 5000 utilisateurs : 500-750 abonnés → **2 450-3 675€/mois**
